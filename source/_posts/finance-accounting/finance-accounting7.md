# 长期经营性资产的报告与解释

## 第一部分：课程整体概括

本节课核心内容是学习企业如何对长期经营性资产（Long-lived Productive Assets）进行会计处理、报告与解释。长期资产是企业运营的基石，其会计处理直接影响利润表和资产负债表。课程逻辑清晰：首先，我们学习如何定义和分类长期资产，并掌握用于分析其使用效率的关键财务比率——**固定资产周转率（Fixed Asset Turnover Ratio）**。其次，重点在于**成本原则（Cost Principle）**的应用，即如何正确地计量和记录资产的初始取得成本以及后续的维护与改良支出。然后，深入探讨资产持有期间的核心会计过程——**成本分摊**，即**折旧（Depreciation）**，并详细讲解了直线法、产量法和双倍余额递减法。接着，课程讨论了当资产价值下跌时如何进行**资产减值（Asset Impairment）**测试与会计处理，以及资产出售或报废时的**处置（Disposal）**会计。最后，课程将概念延伸至没有实物形态的**无形资产（Intangible Assets）**和会逐渐耗竭的**自然资源（Natural Resources）**，介绍了其特殊的计量与摊销/折耗方法。

## 第二部分：核心方法总结与案例

### 1. 固定资产周转率分析
*   **定义/目的**：衡量企业每投入一元固定资产所能产生的销售收入，是评估固定资产使用和管理效率的关键指标。
*   **计算方法**：
    `固定资产周转率 = 销售收入（或营业收入） / 平均固定资产净值`
    *   平均固定资产净值 = （期初固定资产净值 + 期末固定资产净值）/ 2
*   **案例**：FedEx公司2020财年数据。
    *   营业收入 = \$69,217百万
    *   期初固定资产净值 = \$30,429百万，期末 = \$33,608百万
    *   平均净值 = (\$30,429 + \$33,608) / 2 = \$32,018.5百万
    *   固定资产周转率 = \$69,217 / \$32,018.5 ≈ 2.16 次

### 2. 长期资产的初始计量与记录
*   **核心原则**：成本原则。所有为使资产达到预定可使用状态所必需的、合理的支出都应**资本化（Capitalized）**（即计入资产成本）。
*   **资本化成本包含**：买价、运输费、安装调试费、法律费用、销售税等。
*   **维护与改良支出的区分**：
    *   **日常维修（Ordinary Repairs）**：为保持资产原有性能的经常性、小额支出。 → **费用化**，计入当期损益。
    *   **改良（Improvements）**：能延长资产寿命、提高效率或产能的大额、非经常性支出。 → **资本化**，增加资产账面价值。
*   **案例**：FedEx购买飞机。
    *   情景A：现金购买，成本\$26百万。
        *   分录：借：飞机（+A） \$26； 贷：现金（-A） \$26
    *   情景B：部分现金、部分票据购买。飞机成本\$26百万，支付现金\$1百万，余款开立应付票据。
        *   分录：借：飞机（+A） \$26； 贷：现金（-A） \$1， 应付票据（+L） \$25
    *   情景C：以股权加现金购买。发行100万股普通股（每股面值\$1，市价\$10）并支付现金\$16百万，换取飞机（成本\$26百万）。
        *   分录：借：飞机（+A） \$26； 贷：普通股（+SE） \$1， 资本公积（+SE） \$9， 现金（-A） \$16

### 3. 折旧方法
折旧是将固定资产成本在其预计使用寿命内进行系统分摊的过程。不影响资产的市场价值，也非现金支出。

**通用计算所需信息**：资产原值、预计残值（Residual/Salvage Value）、预计使用寿命、折旧方法。

#### **a. 直线法（Straight-Line Method）**
*   **定义**：在资产使用寿命内，每年计提等额的折旧费用。
*   **计算公式**：
    `年折旧额 = (资产原值 - 预计残值) × (1 / 预计使用年限)`
*   **案例**：FedEx购买一辆运输货车，成本\$62,500，预计残值\$2,500，预计使用3年。
    *   年折旧额 = (\$62,500 - \$2,500) × (1/3) = \$20,000
    *   每年末调整分录：借：折旧费用（+E, -SE） \$20,000； 贷：累计折旧（+XA, -A） \$20,000

#### **b. 产量法（Units-of-Production Method）**
*   **定义**：根据资产的实际产出或使用量来分摊折旧成本。
*   **计算公式**：
    1.  计算单位折旧率：`(资产原值 - 预计残值) / 预计总产量（或活动量）`
    2.  计算当期折旧额：`单位折旧率 × 当期实际产量（或活动量）`
*   **案例**：同上货车，预计总行驶里程100,000英里。
    *   单位折旧率 = (\$62,500 - \$2,500) / 100,000英里 = \$0.6/英里
    *   假设第一年行驶30,000英里：折旧额 = \$0.6 × 30,000 = \$18,000
    *   分录：借：折旧费用 \$18,000； 贷：累计折旧 \$18,000

#### **c. 双倍余额递减法（Double Declining-Balance Method）**
*   **定义**：一种加速折旧法，早期多提折旧，后期少提。折旧基数是资产的**账面净值（Net Book Value）**，而非原值减残值。
*   **计算公式**：
    `年折旧额 = 期初账面净值 × (2 / 预计使用年限)`
    *   **注意**：最后一年需进行调整，确保折旧提足后，账面净值等于预计残值。
*   **案例**：同上货车，使用双倍余额递减法。
    *   折旧率 = 2/3
    *   第一年：折旧额 = \$62,500 × (2/3) = \$41,667
    *   第二年：账面净值 = \$62,500 - \$41,667 = \$20,833；折旧额 = \$20,833 × (2/3) = \$13,889
    *   第三年：账面净值 = \$20,833 - \$13,889 = \$6,944。但残值为\$2,500，因此第三年折旧额应为 \$6,944 - \$2,500 = \$4,444。

### 4. 资产减值测试与会计处理
当资产可能发生价值永久性下跌时，需进行减值测试。
*   **测试步骤**：
    1.  **迹象评估**：是否存在导致资产未来现金流减少的事项（如市场衰退、技术过时等）。
    2.  **可回收性测试**：比较资产的**账面净值**与**预计未来现金流量**。若账面净值 > 预计未来现金流量，则资产发生减值。
    3.  **计量减值损失**：`减值损失 = 账面净值 - 资产公允价值（Fair Value）`。将资产账面价值减记至公允价值。
*   **案例**：Delta航空因疫情影响，评估其部分飞机已减值。
    *   账面净值 = \$4,909百万，预计未来现金流 = \$900百万（假设），公允价值 = \$500百万。
    *   可回收性测试：\$4,909 > \$900，发生减值。
    *   减值损失 = \$4,909 - \$500 = \$4,409百万。
    *   分录：借：资产减值损失（+E, -SE） \$4,409； 贷：飞机（-A） \$4,409

### 5. 长期资产处置的会计处理
*   **核心**：计算处置损益，并核销相关资产账户。
    `处置损益 = 处置所得价款（或资产公允价值） - 资产处置时的账面净值`
    *   结果为正 → 利得（Gain）；结果为负 → 损失（Loss）。
*   **案例**：FedEx出售一架飞机。
    *   原值\$30百万，预计使用25年，残值为0，直线法折旧。第17年末出售，售价\$11百万现金。
    *   **步骤1：更新至出售日的折旧**。
        *   年折旧额 = \$30百万 / 25 = \$1.2百万。
        *   截至第17年末，累计折旧 = \$1.2 × 17 = \$20.4百万。
        *   分录（计提第17年折旧）：借：折旧费用 \$1.2； 贷：累计折旧 \$1.2
    *   **步骤2：计算出售日账面净值并确认损益**。
        *   账面净值 = \$30 - \$20.4 = \$9.6百万。
        *   处置利得 = \$11 - \$9.6 = \$1.4百万。
    *   **步骤3：记录处置分录**。
        *   分录：借：现金（+A） \$11.0， 累计折旧（-XA, +A） \$20.4； 贷：飞机（-A） \$30.0， 处置资产利得（+R, +SE） \$1.4

### 6. 无形资产与自然资源
*   **无形资产**：
    *   **摊销（Amortization）**：对于有明确使用寿命的无形资产（如专利、版权），其成本在其使用寿命内分摊，通常用直线法。摊销费用计入利润表。
    *   **不摊销**：对于使用寿命不确定的无形资产（如商标、商誉），不进行摊销，但每年需进行减值测试。
    *   **商誉（Goodwill）**：仅在收购另一企业时产生，计算公式：`商誉 = 收购价格 - 被收购方可辨认净资产公允价值`。不摊销，每年进行减值测试。
*   **自然资源**：
    *   **折耗（Depletion）**：将自然资源的成本在其开采期内进行分摊，常用产量法。折耗金额通常先计入存货成本，待资源出售时再转入销售成本。
        *   分录示例：借：存货（+A） X； 贷：自然资源（-A） X

### 7. 折旧估计变更（补充）
当资产使用寿命或残值的估计发生变更时，采用未来适用法，不调整以前年度折旧。
*   **新折旧额计算公式**：
    `新的年折旧额 = (变更时的账面净值 - 新的预计残值) / 剩余使用寿命`
*   **案例**：FedEx一架飞机原值\$60百万，预计使用20年，残值\$3百万。第5年末，将剩余使用寿命改为21年（总寿命25年），残值改为\$2.4百万。
    *   前4年折旧 = (\$60 - \$3) / 20 = \$2.85百万/年。累计折旧 = \$2.85 × 4 = \$11.4百万。
    *   第5年初账面净值 = \$60 - \$11.4 = \$48.6百万。
    *   新的年折旧额（第5年起）= (\$48.6 - \$2.4) / 21 = \$2.2百万/年。

## M8-4
Computing Book Value (Straight-Line Depreciation)
Calculate the book value of a three-year-old machine that has a cost of \$31,000, an estimated residual value of \$1,000, and an estimated useful life of five years. The company uses straight-line depreciation.

### 解答
- 每年折旧额 = (成本 - 残值) / 使用寿命 = (31,000 - 1,000) / 5 = 30,000 / 5 = 6,000  
- 累计折旧（3年） = 每年折旧额 × 使用年数 = 6,000 × 3 = 18,000  
- 账面价值 = 成本 - 累计折旧 = 31,000 - 18,000 = 13,000  

## M8-5
Calculate the book value of a three-year-old machine that has a cost of \$55,000, an estimated residual value of \$5,000, and an estimated useful life of five years. The company uses double-declining-balance depreciation. Round to the nearest dollar.

### 解答
折旧率 = 2 / 5 = 0.4

第一年折旧 = 0.4 * 55000 = 22000

第一年末账面价值 = 55000 - 22000 = 33000

第二年折旧 = 0.4 * 33000 = 13200

第二年末账面价值 = 33000 - 13200 = 19800

第三年折旧 = 0.4 * 19800 = 7920

第三年末账面价值 = 19800 - 7920 = 11880

账面价值为 \$11,880。

## M8-6
Calculate the book value of a three-year-old machine that has a cost of \$26,000, an estimated residual value of \$1,000, and an estimated useful life of 50,000 machine hours. The company uses units-of-production depreciation and ran the machine 3,200 hours in Year 1; 7,050 hours in Year 2; and 7,500 hours in Year 3.

### 解答
首先，计算折旧基础：成本 \$26,000 - 残值 \$1,000 = \$25,000。

每机器小时折旧率：\$25,000 / 50,000 小时 = \$0.50/小时。

累计折旧：
- 第1年：3,200 小时 × \$0.50 = \$1,600
- 第2年：7,050 小时 × \$0.50 = \$3,525
- 第3年：7,500 小时 × \$0.50 = \$3,750
累计折旧总额：\$1,600 + \$3,525 + \$3,750 = \$8,875

账面价值 = 成本 \$26,000 - 累计折旧 \$8,875 = \$17,125

## E8-3
Shahia Company bought a building for \$382,000 cash and the land on which it was located for \$107,000 cash. The company paid transfer costs of \$9,000 (\$3,000 for the building and \$6,000 for the land). Renovation costs on the building before it could be used were \$21,000.

Required:

1. Give the journal entry to record the purchase of the property, including all relevant expenditures. Assume that all transactions were for cash and that all purchases occurred at the start of the year.

2. Compute straight-line depreciation at the end of one year, assuming an estimated 10-year useful life and a \$15,000 estimated residual value.

3. What would be the net book value of the property (land and building) at the end of Year 2?

### 解答
#### 1.
建筑物成本：购买价 \$382,000 + 转移成本 \$3,000 + 翻新成本 \$21,000 = \$406,000  
土地成本：购买价 \$107,000 + 转移成本 \$6,000 = \$113,000  
总现金支出：\$382,000 + \$107,000 + \$9,000 + \$21,000 = \$519,000  
日记账分录：  
借：建筑物 \$406,000  
借：土地 \$113,000  
贷：现金 \$519,000  

#### 2.
建筑物成本 \$406,000，残值 \$15,000，使用寿命 10 年  
年折旧额 = (\$406,000 - \$15,000) / 10 = \$391,000 / 10 = \$39,100  
一年后直线折旧额为 \$39,100  

#### 3.
土地不折旧，账面净值始终为 \$113,000  
建筑物：  
第一年折旧 \$39,100，累计折旧 \$39,100，账面净值 \$406,000 - \$39,100 = \$366,900  
第二年折旧 \$39,100，累计折旧 \$78,200，账面净值 \$406,000 - \$78,200 = \$327,800  
财产总账面净值：土地 \$113,000 + 建筑物 \$327,800 = \$440,800


## P8-6

Required:  
1. Record journal entries for transactions (a) through (f).

(f) 重新调整资本化

#### 2.  
专利：成本28,000，使用寿命7年，直线法摊销，年摊销额 = 28,000 / 7 = 4,000
